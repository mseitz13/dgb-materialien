# DGB Materialsammlung

Professionelle GitHub-Pages-Website für Unterrichtsmaterialien in Digitaler Grundbildung.

## Veröffentlichung

1. Alle Dateien und Ordner in dieses Repository hochladen.
2. Im Repository zu **Settings → Pages** gehen.
3. Unter **Build and deployment** die Quelle **Deploy from a branch** wählen.
4. Branch **main** und Ordner **/root** auswählen.
5. Speichern.

Die Website ist danach unter folgendem Link erreichbar:

`https://mseitz13.github.io/dgb-materialien/`

## Dateien hinzufügen

PDFs und andere Dokumente am besten in den Ordner `downloads` hochladen.
Anschließend können sie auf den Unterseiten mit einem normalen Link eingebunden werden.

Beispiel:

```html
<a href="../downloads/arbeitsblatt-word.pdf" download>Arbeitsblatt herunterladen</a>
```
